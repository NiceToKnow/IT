Nice To KnowIT Metrics
  • DAU: Daily Active Users
  • WAU: Weekly Active Users
  • MAU: Monthly Active Users
  • CLV / CLTV / LTV / LCV: Customer Lifetime Value
  • NPV / NPW: Net Present Value / Net Present Worth
  • CAC: Customer Acquisition Cost
  • Churn / Churn Rate / Attrition Rate: lost users
  • ROI: Return on Investment
  • Stickiness (the DAU/MAU ratio)
  • User Retention
  • AOV: Average Order Value
  • CR: Conversion Rate
  • RPA: Revenue per acquisition
  • CPA: Cost per acquisition
  • TCR: Task Completion Rate

  • NPS: Net Promoter Score
